# Practice Templates

This directory contains reusable templates for spinning up new projects with our development practices.

## Templates

- **[AGENTS.md.template](AGENTS.md.template)** - Template for AGENTS.md file that directs AI agents on how to work with the project. Copy this to the root of a new project and customize the project-specific sections.

## Using Templates

When starting a new project:

1. Copy templates from this directory to your project root
2. Replace placeholder text (marked with `[...]`)
3. Fill in project-specific context sections
4. Commit as part of Phase 0 (scaffolding)

## Philosophy

Templates are **practices**, not **project artifacts**. They live here to be:
- Reusable across projects
- Evolvable as practices improve
- Shareable as a curriculum

Project-specific execution records (what was actually done) live in `tasks/phase0/` of each project.
